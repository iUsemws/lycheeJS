    1Tool Usage

    Select all frame images using the Select button. Modify the settings to tweak the Sprite, the rest is done automatically.

    Animations are triggered if the filenames contain an underscore.

    Examples: default.png, not-animated.png, animated_01.png, animated_02.png.

    2Engine Usage

    The exported files include the Definition (.js), the Texture (.png) and the Config (.json) for the Entity.

    If you want to use the Entity inside your Project, you have to rename the files accordingly after copying them into the source/entity/ folder.

    Don't forget to modify the lychee.define('game.entity.Entity') line in the .js file accordingly!
